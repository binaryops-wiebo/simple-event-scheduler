claimJobRun

  • claimJobRun(job: Job): Promise<null | Job>

  • Overrides DBAdapter.claimJobRun

    • Defined in src/db/sequelize-adapter.ts:151

    This method attempts to update the lastRunTime of the Job to the current timestamp using the currently known value in the where clause. If no other server running the scheduler has updated the record already, then this update will succeed and the updated Job is returned in the promise. In that case the scheduler will emit an event for the Job. However if the update fails, then no events will be emitted for this job until it is reloaded in the next database load interval.

    Parameters

    • job: Job

    Returns Promise<null | Job>
